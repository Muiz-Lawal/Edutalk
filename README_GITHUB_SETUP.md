## ⚡ 5-Minute Quick Start

```powershell
# 1. Setup git
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

# 2. Create repository at https://github.com/new
# (Name: edutalk, Description provided)

# 3. Copy URL from GitHub, then run:
git remote add origin https://github.com/YOUR-USERNAME/edutalk.git
git branch -M main
git add .
git commit -m "Initial commit: EduTalk platform"
git push -u origin main

# 4. Verify at: https://github.com/YOUR-USERNAME/edutalk
```
