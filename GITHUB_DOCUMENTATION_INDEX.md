## ⚡ Quick Commands

### One-Time Setup

```powershell
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"
```

### Push Project to GitHub

```powershell
git remote add origin https://github.com/YOUR-USERNAME/edutalk.git
git branch -M main
git add .
git commit -m "Initial commit: EduTalk platform"
git push -u origin main
```

### Daily Workflow

```powershell
git pull origin main                          # Get latest
git checkout -b feature/your-feature          # Create branch
git add src/file.js                           # Stage changes
git commit -m "feat: Add feature"             # Commit
git push origin feature/your-feature          # Push
# Create Pull Request on GitHub
```

### Common Checks

```powershell
git status                                    # See status
git log --oneline -10                         # See commits
git diff src/file.js                          # See changes
git check-ignore -v .env                      # Check .gitignore
```

### Documentation Issues

If documentation is unclear:

1. Create a GitHub Issue with `documentation` label
2. Describe what's confusing

### Technical Issues

If you have technical questions:

1. Check the troubleshooting sections
2. Search [Stack Overflow](https://stackoverflow.com)
3. Check [GitHub Community](https://github.community)

### Security Issues

If you find a security vulnerability:

1. Read [SECURITY.md](SECURITY.md#reporting-a-vulnerability)
2. Email security@edutalk.com (or maintainer email)
3. Don't create a public issue
